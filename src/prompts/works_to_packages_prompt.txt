ПРИНЦИПЫ РАСПРЕДЕЛЕНИЯ:

1. АНАЛИЗ ПО КЛЮЧЕВЫМ СЛОВАМ:
   
   ДЕМОНТАЖ → пакеты с "демонтаж", "снос", "разборка":
   - демонтаж, разборка, снос, удаление, очистка
   
   ЭЛЕКТРИКА → пакеты с "электр", "освещение", "проводка":
   - электро, кабель, провод, розетка, выключатель, освещение, щит
   
   САНТЕХНИКА → пакеты с "сантех", "водопровод", "канализация":
   - труба, водопровод, канализация, сантех, смеситель, унитаз
   
   ОТДЕЛКА → пакеты с "отделка", "покраска", "штукатурка":
   - штукатурка, шпаклевка, покраска, обои, плитка, облицовка
   
   ПОЛЫ → пакеты с "пол", "напольные", "покрытие":
   - стяжка, ламинат, паркет, линолеум, напольное покрытие
   
   КОНСТРУКЦИИ → пакеты с "монтаж", "установка", "конструкции":
   - перегородка, каркас, конструкция, балка, колонна

2. АНАЛИЗ ПО КОДАМ РАБОТ:
   - Коды серии 01-09: подготовительные и земляные работы
   - Коды серии 10-15: конструкции и фундаменты  
   - Коды серии 16-20: электромонтажные работы
   - Коды серии 21-25: сантехнические работы
   - Коды серии 26-30: отделочные работы

3. ЛОГИКА ВЫБОРА:
   - Ищи наиболее близкий по смыслу пакет
   - При сомнении выбирай пакет с более общим названием
   - Если работа может подойти к нескольким пакетам, выбирай по приоритету:
     1. Точное совпадение по ключевым словам
     2. Совпадение по типу работ (демонтаж, монтаж, отделка)
     3. Наиболее общий подходящий пакет

4. СПЕЦИАЛЬНЫЕ СЛУЧАИ:
   - Материалы обычно НЕ включаются в пакеты работ
   - Если в названии есть "материал", "поставка" - назначай в ближайший пакет работ
   - Транспортные работы - в общестроительные пакеты
   - Проектные работы - в соответствующие технические пакеты
